Welcome to NorthStar!

The NorthStar web portal enables ThingSpace users to access general purpose data processing and analysis capabilities that can be used to analyze IoT device data and extract useful and sharable results.

Learn more

Notebooks

Notebooks enable users to present data using a number of visualizations. Each visualization can be from a single source of data or aggregated values from multiple sources. Visualizations can be populated with dynamic queries created by the specific notebook as well as by accepting input from the user.

View details »

Transformations

Transformations enable users to define code that can be executed on a device event (e.g., process device data) or on a configured schedule.
